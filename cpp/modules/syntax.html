Składnia i Operatory ⚙️

Podstawy składni C++ i wszystkie operatory.

1. Podstawowa Składnia

  • Każda instrukcja kończy się średnikiem ;
  • Bloki kodu w nawiasach klamrowych { }
  • Wielkość liter ma znaczenie (main ≠ Main)
  • Komentarze: // (jedna linia) lub /* */ (wiele linii)
#include <iostream>
using namespace std;

int main() {
    // Komentarz jednowierszowy
    cout << "Hello!" << endl;
    
    /* Komentarz
       wielowierszowy */
    
    return 0;  // Średnik obowiązkowy!
}

4. Operatory Logiczne

OperatorNazwaPrzykładWynik

||OR (lub)true || falsetrue

int wiek = 25;
bool maPrawo = true;

# Czy może prowadzić? (wiek >= 18 I ma prawo)
if (wiek >= 18 && maPrawo) {
    cout << "Może prowadzić" << endl;
}

# Czy jest dzieckiem LUB seniorem?
if (wiek < 18 || wiek > 65) {
    cout << "Ulga!" << endl;
}

5. Wejście od Użytkownika (cin)

#include <iostream>
#include <string>
using namespace std;

int main() {
    string imie;
    int wiek;
    
    cout << "Podaj imię: ";
    cin >> imie;  # cin odczytuje do pierwszej spacji!
    
    cout << "Podaj wiek: ";
    cin >> wiek;
    
    cout << "Cześć " << imie << ", masz " << wiek << " lat!" << endl;
    
    # Dla całych linii użyj getline:
    string pelneImie;
    cout << "Podaj pełne imię: ";
    getline(cin, pelneImie);  # Czyta całą linię ze spacjami
    
    return 0;
}
