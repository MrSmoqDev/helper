Wstęp do C++ 🚀

Szybki, wydajny i potężny język programowania.

1. Czym jest C++?

C++ to język programowania ogólnego zastosowania, stworzony jako rozszerzenie języka C. Jest znany z:

  • ✅ Wysokiej wydajności – kod kompilowany do natywnego
  • ✅ Kontroli pamięci – ręczne zarządzanie zasobami
  • ✅ Programowania obiektowego – klasy, dziedziczenie, polimorfizm
  • ✅ Szerokiego zastosowania – gry, systemy, aplikacje

2. Pierwszy Program – Hello World

#include <iostream>

int main() {
    std::cout << "Hello World!" << std::endl;
    return 0;
}

Wyjaśnienie:

  • #include <iostream> – dołącza bibliotekę wejścia/wyjścia
  • int main() – funkcja główna, od niej zaczyna się program
  • std::cout – strumień wyjścia (wyświetlanie)
  • std::endl – nowa linia + flush bufora
  • return 0 – program zakończył się sukcesem

3. Kompilacja i Uruchomienie

C++ jest językiem kompilowanym – kod źródłowy musi zostać przetłumaczony na język maszynowy.

🔹 Windows (MinGW/g++)

g++ -o program main.cpp
program.exe

🔹 Linux/Mac

g++ -o program main.cpp
./program

🔹 Visual Studio

Ctrl+F5 – Uruchom bez debugowania
F5 – Uruchom z debugowaniem

4. Struktura Programu C++

// Dyrektywy preprocesora
#include <iostream>
#include <string>

// Przestrzeń nazw (umożliwia pominięcie std::)
using namespace std;

// Funkcja główna
int main() {
    // Twój kod tutaj
    cout << "Witaj w C++!" << endl;
    return 0;
}
