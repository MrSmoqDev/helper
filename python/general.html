Wstęp do Pythona

Podstawy składni i najważniejsze informacje na start.

Dlaczego Python?

Python jest językiem interpretowanym, co oznacza, że kod jest wykonywany linia po linii. Jest niezwykle czytelny, co sprawia, że idealnie nadaje się dla początkujących.

Jak uruchomić kod?

Możesz użyć IDLE, VS Code lub uruchomić skrypt w terminalu:

python main.py

Wybierz konkretną bibliotekę

Kliknij w sidebarze po lewej stronie, aby przejść do szczegółowego opisu konkretnych modułów, np. Math.
