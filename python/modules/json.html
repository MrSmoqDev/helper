JSON i Konfiguracje 📦

Zapisywanie i wczytywanie danych w formacie JSON – idealne do configów, save'ów i API.

1. Czym jest JSON?

JSON (JavaScript Object Notation) to lekki format wymiany danych. Wygląda jak słownik Pythona, ale jest tekstem – dzięki czemu łatwo go zapisać do pliku lub wysłać przez internet.

# Przykład JSON (jako tekst)
{
  "gracz": "Jan",
  "poziom": 15,
  "ekwipunek": ["miecz", "tarcza"],
  "aktywny": true
}

2. Moduł json – Podstawy

Python ma wbudowany moduł json do konwersji między obiektami Pythona a JSON-em.

FunkcjaOpisPrzykład
json.dumps(obj)Python → JSON (string)dumps({"a":1}) → '{"a": 1}'
json.loads(str)JSON (string) → Pythonloads('{"a":1}') → {"a":1}
json.dump(obj, f)Zapisz JSON do plikudump(data, open("f.json","w"))
json.load(f)Wczytaj JSON z plikuload(open("f.json"))
import json

# Python → JSON (string)
dane = {"imie": "Anna", "wiek": 25, "aktywna": True}
json_str = json.dumps(dane, ensure_ascii=False, indent=2)
print(json_str)

# JSON (string) → Python
wczytane = json.loads(json_str)
print(wczytane["imie"])  # "Anna"

3. Zapis i Odczyt z Pliku

import json

# --- ZAPIS ---
konfiguracja = {
    "nazwa_gry": "Moja Przygoda",
    "glosnosc": 0.7,
    "trudnosc": "średnia",
    "ostatni_zapis": "2024-01-15"
}

with open("config.json", "w", encoding="utf-8") as f:
    json.dump(konfiguracja, f, ensure_ascii=False, indent=2)
print("✅ Zapisano config.json")

# --- ODCZYT ---
try:
    with open("config.json", "r", encoding="utf-8") as f:
        wczytany_config = json.load(f)
    print(f"🎮 Gra: {wczytany_config['nazwa_gry']}")
    print(f"🔊 Głośność: {wczytany_config['glosnosc']*100:.0f}%")
except FileNotFoundError:
    print("❌ Brak pliku config.json – używam domyślnych")

4. 🛠️ Projekt: Menedżer Ustawień

import json
import os

PLIK_CONFIG = "ustawienia.json"

def wczytaj_ustawienia():
    """Wczytuje ustawienia z pliku lub zwraca domyślne"""
    domyslne = {
        "jezyk": "pl",
        "tryb_ciemny": True,
        "powiadomienia": True,
        "wersja": "1.0"
    }
    
    if not os.path.exists(PLIK_CONFIG):
        zapisz_ustawienia(domyslne)
        return domyslne
    
    with open(PLIK_CONFIG, "r", encoding="utf-8") as f:
        return json.load(f)

def zapisz_ustawienia(ust):
    """Zapisuje ustawienia do pliku"""
    with open(PLIK_CONFIG, "w", encoding="utf-8") as f:
        json.dump(ust, f, ensure_ascii=False, indent=2)

def zmien_ustawienie(klucz, wartosc):
    """Zmienia pojedyncze ustawienie i zapisuje"""
    ust = wczytaj_ustawienia()
    ust[klucz] = wartosc
    zapisz_ustawienia(ust)
    print(f"✅ {klucz} = {wartosc}")

# Demo
ust = wczytaj_ustawienia()
print(f"🌐 Język: {ust['jezyk']}")
zmien_ustawienie("glosnosc", 0.8)
print("✅ Gotowe!")

5. JSON a API – Przykład

Większość API internetowych zwraca dane w JSON. Oto jak to wygląda w praktyce:

import json

# Symulacja odpowiedzi z API
odpowiedz_api = '''
{
  "status": "success",
  "data": {
    "users": [
      {"id": 1, "name": "Anna"},
      {"id": 2, "name": "Jan"}
    ],
    "count": 2
  }
}
'''

# Parsujemy JSON
dane = json.loads(odpowiedz_api)

# Pracujemy na danych
if dane["status"] == "success":
    for user in dane["data"]["users"]:
        print(f"👤 {user['id']}: {user['name']}")

📝 Ściąga JSON

FunkcjaKierunekUwagi
dumps(obj)Python → strensure_ascii=False dla polskich znaków
loads(str)str → PythonWymaga poprawnego JSON-a
dump(obj, f)Python → plikindent=2 dla czytelności
load(f)plik → PythonZawsze używaj with open()

Typy Python → JSON:

  • dict → {}
  • list/tuple → []
  • str → "tekst"
  • int/float → liczba
  • True/False/None → true/false/null
