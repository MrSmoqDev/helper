Moduł Math 📐

Kompletny przewodnik po matematyce w Pythonie – od podstaw po kalkulator.

1. Co to jest moduł Math?

Moduł math to wbudowana biblioteka Pythona, która dostarcza zaawansowane funkcje matematyczne zdefiniowane przez standard C. Nie trzeba nic instalować – jest dostępny od razu po zainstalowaniu Pythona.

import math  # To musi być na początku pliku

2. Podstawowe Operacje Arytmetyczne

Python obsługuje wszystkie podstawowe działania matematyczne bez importowania modułu math. Oto kompletna lista operatorów:

Operator Nazwa Przykład Wynik
+ Dodawanie 5 + 3 8
- Odejmowanie 10 - 4 6
* Mnożenie 6 * 7 42
/ Dzielenie (float) 10 / 3 3.333...
// Dzielenie całkowite 10 // 3 3
% Modulo (reszta) 10 % 3 1
** Potęgowanie 2 ** 3 8 
# Przykłady użycia
print(2 + 2)           # 4
print(10 - 3)          # 7
print(4 * 5)           # 20
print(15 / 2)          # 7.5 (zawsze float!)
print(15 // 2)         # 7 (tylko część całkowita)
print(15 % 2)          # 1 (reszta z dzielenia)
print(2 ** 4)          # 16 (2 do potęgi 4)

# Kolejność działań jak w matematyce
print(2 + 3 * 4)       # 14 (najpierw mnożenie!)
print((2 + 3) * 4)     # 20 (najpierw nawias)
💡 Tip: Python automatycznie konwertuje wyniki do float przy dzieleniu /. Użyj int() lub //, jeśli chcesz wynik całkowity.

3. Ważne Stałe Matematyczne

Math udostępnia precyzyjne wartości stałych, których nie musisz wpisywać ręcznie.

Stała Opis Wartość
math.pi Liczba Pi (π) 3.14159...
math.e Liczba Eulera (e) 2.71828...
math.tau Stała Tau (2π) 6.28318... 
import math
print(f"Pi: {math.pi}")
print(f"E: {math.e}")

4. Zaokrąglanie Liczb

To najczęściej używane funkcje. Zwróć uwagę na różnicę między floor a ceil.

  • math.floor(x) – Zaokrągla w dół (do podłogi).
  • math.ceil(x) – Zaokrągla w górę (do sufitu).
  • math.trunc(x) – Ucina część dziesiętną (jak int).
import math

liczba = 3.7

# Podłoga (zawsze mniej lub równo)
print(math.floor(liczba))  # Wynik: 3

# Sufit (zawsze więcej lub równo)
print(math.ceil(liczba))   # Wynik: 4

# Ucięcie (sama część całkowita)
print(math.trunc(-3.7))    # Wynik: -3

5. Potęgi i Pierwiastki

Do obliczeń algebraicznych używamy tych funkcji:

  • math.sqrt(x) – Pierwiastek kwadratowy.
  • math.pow(x, y) – Potęgowanie (x do potęgi y).
  • math.factorial(x) – Silnia (x!).
import math

# Pierwiastek z 16
print(math.sqrt(16))        # Wynik: 4.0

# 2 do potęgi 3 (2*2*2)
print(math.pow(2, 3))       # Wynik: 8.0

# Silnia z 5 (5*4*3*2*1)
print(math.factorial(5))    # Wynik: 120

6. Trygonometria (Ważne!)

Funkcje trygonometryczne w Pythonie pracują na radianach, a nie stopniach!

  • math.sin(x), math.cos(x), math.tan(x)
  • math.degrees(x) – Zamienia radiany na stopnie.
  • math.radians(x) – Zamienia stopnie na radiany.
import math

kat_stopnie = 90

# Najpierw zamień stopnie na radiany!
kat_radiany = math.radians(kat_stopnie)

# Teraz oblicz sinus
wynik = math.sin(kat_radiany)

print(f"Sinus 90 stopni: {wynik}") # Wynik: 1.0

7. 🧮 Poradnik: Prosty Kalkulator (Wiele Liczb)

Chcesz zrobić kalkulator, który obsłuży działanie typu 2+2+3+4? Oto jak to zrobić krok po kroku!

🔹 Krok 1: Pobranie danych od użytkownika

# Pobieramy całe wyrażenie jako tekst
wyrazenie = input("Wpisz działanie (np. 2+2+3): ")
print(f"Obliczam: {wyrazenie}")

🔹 Krok 2: Rozdzielenie liczb i operatorów

Najprostszy sposób to użycie funkcji eval() – ale uwaga, jest niebezpieczna przy danych od użytkowników! Do nauki OK, do produkcji – lepiej parsować ręcznie.

# ⚠️ TYLKO DO NAUKI - eval() wykonuje dowolny kod Python!
wynik = eval(wyrazenie)
print(f"Wynik: {wynik}")

🔹 Krok 3: Bezpieczna wersja (tylko + i -)

Lepsze podejście – ręczne parsowanie tylko dodawania i odejmowania:

def kalkulator_prosty(wyrazenie):
    """Oblicza wyrażenie z + i - (bez eval!)"""
    # Usuwamy spacje
    wyrazenie = wyrazenie.replace(" ", "")
    
    # Zaczynamy od pierwszej liczby
    wynik = 0
    aktualna_liczba = ""
    znak = "+"
    
    for znak_wyrazenia in wyrazenie + "+":  # + na końcu, żeby przetworzyć ostatnią liczbę
        if znak_wyrazenia in "+-":
            if aktualna_liczba:
                liczba = float(aktualna_liczba)
                if znak == "+":
                    wynik += liczba
                else:
                    wynik -= liczba
                aktualna_liczba = ""
            znak = znak_wyrazenia
        else:
            aktualna_liczba += znak_wyrazenia
    
    return wynik

# Test
print(kalkulator_prosty("2+2+3+4"))    # Wynik: 11.0
print(kalkulator_prosty("10-3-2"))      # Wynik: 5.0

8. 🎛️ Kalkulator z Menu i Obsługą Błędów

Teraz połączmy wszystko w program z menu, który pozwala wybrać działanie i wprowadzić wiele liczb:

def pokaz_menu():
    print("\n=== 🧮 KALKULATOR 🧮")
    print("1. ➕ Dodawanie")
    print("2. ➖ Odejmowanie")
    print("3. ✖️ Mnożenie")
    print("4. ➗ Dzielenie")
    print("5. 🚪 Wyjdź")

def pobierz_liczby():
    """Pobiera wiele liczb od użytkownika"""
    print("\nWpisz liczby oddzielone spacją (np. 2 4 6 8):")
    wejscie = input("> ")
    
    try:
        # Dzielimy tekst i zamieniamy na float
        liczby = [float(x) for x in wejscie.split()]
        return liczby
    except ValueError:
        print("❌ Błąd: Wpisz tylko liczby!")
        return []

def dodawanie(liczby):
    wynik = sum(liczby)  # sum() to wbudowana funkcja!
    print(f"✅ Wynik: {' + '.join(map(str, liczby))} = {wynik}")

def odejmowanie(liczby):
    wynik = liczby[0]
    for l in liczby[1:]:
        wynik -= l
    print(f"✅ Wynik: {' - '.join(map(str, liczby))} = {wynik}")

def mnozenie(liczby):
    wynik = 1
    for l in liczby:
        wynik *= l
    print(f"✅ Wynik: {' × '.join(map(str, liczby))} = {wynik}")

def dzielenie(liczby):
    if len(liczby) < 2:
        print("❌ Potrzebne co najmniej 2 liczby!")
        return
    
    wynik = liczby[0]
    try:
        for l in liczby[1:]:
            wynik /= l
        print(f"✅ Wynik: {' ÷ '.join(map(str, liczby))} = {wynik:.4f}")
    except ZeroDivisionError:
        print("❌ Nie można dzielić przez zero!")

def main():
    while True:
        pokaz_menu()
        wybor = input("\nWybierz opcję (1-5): ")
        
        if wybor == "5":
            print("👋 Do zobaczenia!")
            break
        
        if wybor not in ["1", "2", "3", "4"]:
            print("❌ Nieprawidłowy wybór!")
            continue
        
        liczby = pobierz_liczby()
        if not liczby:
            continue
        
        if wybor == "1":
            dodawanie(liczby)
        elif wybor == "2":
            odejmowanie(liczby)
        elif wybor == "3":
            mnozenie(liczby)
        elif wybor == "4":
            dzielenie(liczby)

if __name__ == "__main__":
    main()

9. 🚀 Zaawansowany: Kalkulator z Pętlą i Komendami

Chcesz kalkulator, gdzie wpisujesz 2 + 2 + 3 = i dostajesz wynik? Oto wersja z pętlą wprowadzania:

def kalkulator_interaktywny():
    """Kalkulator z pętlą: wpisuj liczby i operatory po kolei"""
    wynik = None
    
    print("\n=== 🧮 KALKULATOR INTERAKTYWNY 🧮")
    print("Wpisuj: liczba, potem operator (+,-,*,/), na koniec '=' aby zobaczyć wynik")
    print("Wpisz 'exit' aby wyjść\n")
    
    while True:
        wejscie = input(f"{wynik if wynik else '0'} > ").strip()
        
        if wejscie.lower() == "exit":
            print("👋 Koniec.")
            break
        
        # Jeśli to pierwsza liczba
        if wynik is None:
            try:
                wynik = float(wejscie)
                print(f"✓ {wynik}")
            except ValueError:
                print("❌ Wpisz liczbę!")
            continue
        
        # Jeśli to operator
        if wejscie in ["+", "-", "*", "/", "="]:
            operator = wejscie
            
            if operator == "=":
                print(f"🎯 WYNIK: {wynik}")
                wynik = None  # Reset
                continue
            
            # Pobierz kolejną liczbę
            liczba_wejscie = input(f"{wynik} {operator} ").strip()
            
            try:
                liczba = float(liczba_wejscie)
                
                if operator == "+":
                    wynik += liczba
                elif operator == "-":
                    wynik -= liczba
                elif operator == "*":
                    wynik *= liczba
                elif operator == "/":
                    if liczba == 0:
                        print("❌ Nie można dzielić przez zero!")
                        continue
                    wynik /= liczba
                
                print(f"✓ {wynik}")
                
            except ValueError:
                print("❌ Błąd: Wpisz liczbę!")
                continue
        else:
            print("❌ Wpisz operator (+, -, *, /) lub =")

# Uruchom
kalkulator_interaktywny()
