Powrót do Menu

Prosta Gra Konsolowa 🎮

Naucz się tworzyć interaktywne gry w terminalu z menu i pętlami.

1. Co będziemy tworzyć?

Stworzymy klasyczną grę "Zgadnij Liczbę". Komputer losuje liczbę od 1 do 100, a Ty musisz ją odgadnąć. Po każdej próbie otrzymasz podpowiedź, czy szukana liczba jest większa, czy mniejsza.

Czego się nauczysz:

  • ✅ Tworzenie menu w konsoli
  • ✅ Pętla while (główna pętla gry)
  • ✅ Obsługa błędów (try-except)
  • ✅ Losowanie liczb (random)
  • ✅ Funkcje i modularność kodu

2. Pełny Kod Gry

Najpierw zobaczmy całość. Skopiuj ten kod do pliku np. gra.py i uruchom.

import random
import time

# --- FUNKCJE POMOCNICZE ---

def wyswietl_menu():
    """Wyświetla główne menu gry"""
    print("\n=== 🎮 ZGADNIJ LICZBĘ 🎮 ===")
    print("1. 🎲 Graj")
    print("2. ℹ️  Instrukcja")
    print("3. 🚪 Wyjdź")
    print("============================")

def graj():
    """Główna logika gry"""
    liczba = random.randint(1, 100)
    proba = 0
    
    print("\n🤔 Wylosowałem liczbę od 1 do 100!")
    
    while True:
        try:
            guess = int(input("\nTwoja liczba: "))
            proba += 1
            
            if guess < liczba:
                print("📈 Za mało! Spróbuj większej.")
            elif guess > liczba:
                print("📉 Za dużo! Spróbuj mniejszej.")
            else:
                print(f"🎉 BRAWO! Zgadłeś w {proba} próbach!")
                break
                
        except ValueError:
            print("❌ To nie jest liczba! Wpisz cyfry.")

def instrukcja():
    """Wyświetla instrukcję"""
    print("\n📖 INSTRUKCJA:")
    print("Komputer losuje liczbę 1-100.")
    print("Masz podpowiedzi (za mało/za dużo).")
    print("Wygraj w jak najmniejszej liczbie prób!")
    time.sleep(3)

# --- GŁÓWNA PĘTLA PROGRAMU ---

def main():
    """Główna pętla menu"""
    while True:
        wyswietl_menu()
        wybor = input("\nWybierz opcję (1-3): ")
        
        if wybor == "1":
            graj()
        elif wybor == "2":
            instrukcja()
        elif wybor == "3":
            print("\n👋 Dziękuję za grę! Do zobaczenia.")
            break
        else:
            print("❌ Nieprawidłowy wybór. Spróbuj ponownie.")

# Uruchomienie programu
if __name__ == "__main__":
    main()

3. Analiza Kodu - Funkcje

Kod podzieliliśmy na funkcje, żeby był czytelny. Każda funkcja odpowiada za jedno zadanie.

🔹 wyswietl_menu()

Wyświetla opcje do wyboru. Używamy print z emoji, żeby menu wyglądało nowocześnie.

🔹 graj()

To serce gry. Zawiera:

  • random.randint(1, 100) – losuje liczbę.
  • Pętlę while True – gra trwa, dopóki gracz nie zgadnie.
  • try-except – chroni przed wpisaniem tekstu zamiast liczby.

🔹 main()

Główna pętla programu. Działa w nieskończoność, dopóki użytkownik nie wybierze opcji "Wyjdź".

4. Jak działa pętla menu?

To najważniejszy wzór w programowaniu konsolowym. Zobacz schemat:

while True:          # 1. Pętla nieskończona
    pokaz_menu()     # 2. Wyświetl opcje
    wybor = input()  # 3. Pobierz wybór
    
    if wybor == "1":
        akcja_1()    # 4. Wykonaj akcję
    elif wybor == "2":
        akcja_2()
    elif wybor == "3":
        break         # 5. Wyjdź z pętli

Dlaczego while True? Bo chcemy, żeby menu pojawiało się po każdej zakończonej grze, a nie tylko raz.

5. Obsługa Błędów (Try-Except)

Gdy użytkownik wpisze "abc" zamiast liczby, program by się crashował. try-except to предотвращa.

try:
    guess = int(input("Twoja liczba: "))
    # Kod wykonuje się, jeśli input to liczba
    
except ValueError:
    print("❌ To nie jest liczba!")
    # Kod wykonuje się, jeśli wystąpił błąd

Dzięki temu gra działa dalej, nawet jeśli gracz popełni błąd.
