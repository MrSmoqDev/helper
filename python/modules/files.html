Praca z Plikami 📁

Naucz się zapisywać, odczytywać i przetwarzać dane z plików tekstowych.

1. Dlaczego pliki są ważne?

Zmienne w Pythonie są tymczasowe – znikają po zamknięciu programu. Pliki pozwalają trwale zapisać dane, które możesz wczytać później. To podstawa tworzenia:

  • ✅ Dzienników (logów) z działania programu
  • ✅ Zapisów stanu gry (save'y)
  • ✅ Konfiguracji aplikacji
  • ✅ Eksportu danych (raporty, listy)

2. Tryby Otwierania Plików

Przed pracą z plikiem musisz określić, co chcesz z nim zrobić. Oto najważniejsze tryby:

Tryb Nazwa Opis Ryzyko
"r" Read (Odczyt) Tylko czytanie. Plik musi istnieć. ❌ Błąd jeśli brak pliku
"w" Write (Zapis) Zapis (nadpisuje!). Tworzy plik jeśli nie ma. ⚠️ Kasuje całą zawartość!
"a" Append (Dopisz) Dopisuje na końcu pliku. Nie kasuje danych. ✅ Bezpieczny
"r+" Read+Write Odczyt i zapis jednocześnie. ⚠️ Wymaga ostrożności
⚠️ UWAGA: Tryb "w" jest niebezpieczny! Jeśli otworzysz istniejący plik w tym trybie, cała jego zawartość zostanie natychmiast usunięta.

3. Instrukcja with (Best Practice)

Nigdy nie używaj samego open() bez close(). Najbezpieczniejszą metodą jest użycie menedżera kontekstu with.

❌ ŹLE (Stara metoda)

f = open("plik.txt", "r")
dane = f.read()
f.close()  # Możesz zapomnieć!

✅ DOBRZE (Menedżer)

with open("plik.txt", "r") as f:
    dane = f.read()
# Plik zamyka się sam!

Zalety with:

  • Plik zamyka się automatycznie (nawet przy błędzie)
  • Kod jest krótszy i czytelniejszy
  • Mniejsze ryzyko wycieku pamięci

4. Zapisywanie do Pliku

Używamy trybu "w" (nadpisz) lub "a" (dopisz). Pamiętaj o znakach nowej linii \n.

# --- Zapis (nadpisanie) ---
with open("notatki.txt", "w", encoding="utf-8") as plik:
    plik.write("Pierwsza linia\n")
    plik.write("Druga linia\n")
    # \n to znak nowej linii (Enter)

# --- Dopisywanie (bezpieczne) ---
with open("notatki.txt", "a", encoding="utf-8") as plik:
    plik.write("Trzecia linia (dopisana)\n")
    # Poprzednie dane zostają!
💡 encoding="utf-8": Zawsze dodawaj ten parametr, jeśli używasz polskich znaków (ą, ę, ś, ć). Bez tego możesz dostać krzaczki zamiast tekstu.

5. Odczytywanie z Pliku

Masz trzy główne metody czytania zawartości pliku:

Metoda Opis Zwraca
read() Czyta cały plik naraz Jeden długi string
readline() Czyta jedną linię String (jedna linia)
readlines() Czyta wszystkie linie Lista stringów 
# --- Metoda 1: Cały plik ---
with open("notatki.txt", "r", encoding="utf-8") as f:
    calosc = f.read()
    print(calosc)

# --- Metoda 2: Linia po linii (najlepsza dla dużych plików) ---
with open("notatki.txt", "r", encoding="utf-8") as f:
    for linia in f:
        print(linia.strip())  # .strip() usuwa \n z końca

# --- Metoda 3: Do listy ---
with open("notatki.txt", "r", encoding="utf-8") as f:
    lista = f.readlines()
    print(lista[0])  # Pierwsza linia

6. Praktyczny Przykład: Dziennik (Logger)

Stwórzmy prosty system logowania zdarzeń do pliku. Każde uruchomienie programu dopisze nową linię z datą.

import time

def zapisz_log(wiadomosc):
    """Dopisuje wiadomość do pliku log.txt z datą"""
    timestamp = time.strftime("%Y-%m-%d %H:%M:%S")
    
    with open("log.txt", "a", encoding="utf-8") as plik:
        plik.write(f"[{timestamp}] {wiadomosc}\n")
    
    print("✓ Zapisano do log.txt")

def wczytaj_log():
    """Wyświetla całą zawartość logu"""
    try:
        with open("log.txt", "r", encoding="utf-8") as plik:
            print("\n=== ZAWARTOŚĆ LOGU ===")
            print(plik.read())
    except FileNotFoundError:
        print("❌ Plik log.txt nie istnieje jeszcze.")

# --- Test ---
zapisz_log("Program został uruchomiony")
zapisz_log("Użytkownik kliknąc przycisk")
wczytaj_log()

Wynik w pliku log.txt:

[2024-01-15 10:30:00] Program został uruchomiony
[2024-01-15 10:30:05] Użytkownik kliknąc przycisk

7. Zastosowanie: Zapis Gry (Save/Load)

Oto jak możesz zapisać postęp gracza do pliku i wczytać go później:

def zapisz_gre(imie, poziom, punkty):
    """Zapisuje stan gry do pliku"""
    with open("save.txt", "w", encoding="utf-8") as f:
        f.write(f"{imie}\n{poziom}\n{punkty}\n")
    print("💾 Gra zapisana!")

def wczytaj_gre():
    """Wczytuje stan gry z pliku"""
    try:
        with open("save.txt", "r", encoding="utf-8") as f:
            linie = f.readlines()
            imie = linie[0].strip()
            poziom = int(linie[1].strip())
            punkty = int(linie[2].strip())
            return imie, poziom, punkty
    except:
        return None

# Użycie
zapisz_gre("Jan", 5, 1500)
dane = wczytaj_gre()
if dane:
    print(f"Gracz: {dane[0]}, Poziom: {dane[1]}")

8. Obsługa Błędów przy Plikach

Najczęstsze błędy i jak ich unikać:

Błąd Przyczyna Rozwiązanie
FileNotFoundError Plik nie istnieje (tryb "r") try-except lub sprawdź przed otwarciem
PermissionError Brak uprawnień do pliku Sprawdź czy plik nie jest otwarty w innym programie
UnicodeDecodeError Problem z kodowaniem znaków Zawsze używaj encoding="utf-8" 
try:
    with open("dane.txt", "r", encoding="utf-8") as f:
        dane = f.read()
except FileNotFoundError:
    print("❌ Plik nie istnieje. Tworzę nowy...")
    with open("dane.txt", "w", encoding="utf-8") as f:
        f.write("")
except PermissionError:
    print("❌ Brak uprawnień do pliku!")
