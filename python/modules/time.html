Moduł Time ⏳

Praca z czasem, opóźnieniami i datami w Pythonie.

1. Wstęp do modułu Time

Moduł time pozwala na operacje związane z czasem: wstrzymywanie programu, pobieranie aktualnej godziny, mierzenie wydajności kodu czy formatowanie dat.

import time  # Import na początku pliku

2. Wstrzymywanie Programu (Sleep)

Funkcja time.sleep() zatrzymuje wykonywanie kodu na określoną liczbę sekund. Przydaje się np. przy tworzeniu pętli, odliczaniu lub opóźnianiu zapytań do API.

import time

print("Start...")
time.sleep(2)  # Czekaj 2 sekundy
print("Koniec po 2 sekundach!")

Zastosowanie: Odliczanie, animacje w konsoli, limitowanie zapytań.

3. Aktualny Czas i Timestamp

Python przechowuje czas jako timestamp – liczbę sekund, która minęła od 1 stycznia 1970 roku (tzw. Epoka Uniksa).

  • time.time() – Zwraca aktualny timestamp (liczba float).
  • time.ctime() – Zamienia timestamp na czytelną datę (string).
  • time.localtime() – Zwraca obiekt z rozpakowanymi danymi (rok, miesiąc, dzień...).
import time

# Timestamp (sekundy od 1970)
teraz_ts = time.time()
print(f"Timestamp: {teraz_ts}")

# Czytelna data
print(time.ctime())  # np. Mon Oct 23 15:30:00 2023

# Szczegółowy obiekt czasu
lokalny = time.localtime()
print(f"Rok: {lokalny.tm_year}")
print(f"Godzina: {lokalny.tm_hour}")

4. Mierzenie Czasu Wykonania

To jedna z najważniejszych funkcji w praktyce. Pozwala sprawdzić, jak szybki jest Twój kod (np. do optymalizacji algorytmów).

import time

start = time.time()  # Zapisz czas startu

--- Twój kod do sprawdzenia ---
time.sleep(1)
-----------------------------

koniec = time.time()  # Zapisz czas końca

trwanie = koniec - start
print(f"Kod wykonywał się: {trwanie:.4f} sekundy")

5. Formatowanie Daty (strftime)

Funkcja time.strftime() pozwala sformatować czas według własnego wzoru. Używamy specjalnych symboli (dyrektyw).

Kod Znaczenie Przykład
%Y Rok (4 cyfry) 2023
%m Miesiąc (01-12) 10
%d Dzień miesiąca (01-31) 23
%H Godzina (00-23) 15
%M Minuta (00-59) 30
%S Sekunda (00-59) 45 
import time

teraz = time.localtime()

# Format: DD/MM/YYYY
data = time.strftime("%d/%m/%Y", teraz)
print(f"Data: {data}")  # Data: 23/10/2023

# Format: Godzina:Minuta
godzina = time.strftime("%H:%M", teraz)
print(f"Godzina: {godzina}")  # Godzina: 15:30

6. Praktyczny Przykład: Odliczanie

Połączenie wiedzy o sleep, strftime i pętlach. Stwórzmy prosty zegar odliczający.

import time
import os

def odliczanie(sekundy):
    while sekundy > 0:
        # Wyczyszczenie konsoli (działa na Windows/Linux)
        os.system('cls' if os.name == 'nt' else 'clear')
        
        print(f"Pozostało: {sekundy} s")
        time.sleep(1)
        sekundy -= 1
    
    print("⏰ KONIEC! ⏰")

odliczanie(5)
