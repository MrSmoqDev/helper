Praca z Tekstem 🔤

Manipulacja napisami – czyszczenie, formatowanie, wyszukiwanie.

1. Dlaczego tekst jest ważny?

Większość danych w programowaniu to tekst: input użytkownika, pliki konfiguracyjne, odpowiedzi API. Umiejętność obróbki stringów to podstawa.

tekst = "  Python Jest Super!  "

2. Podstawowe Metody Stringa

MetodaOpisPrzykład
.lower()Małe litery"ABC".lower() → "abc"
.upper()Wielkie litery"abc".upper() → "ABC"
.strip()Usuwa spacje z końców" hi ".strip() → "hi"
.replace(a,b)Zamienia a na b"kot".replace("o","a") → "kat"
.split(sep)Dzieli string na listę"a,b,c".split(",") → ["a","b","c"]
.join(lista)Łączy listę w string"-".join(["a","b"]) → "a-b"
tekst = "  Python Jest Super!  "

print(tekst.lower())      # "  python jest super!  "
print(tekst.strip())       # "Python Jest Super!"
print(tekst.replace(" ", "_")) # "__Python_Jest_Super!__"
print(tekst.split())        # ["Python", "Jest", "Super!"]

3. Formatowanie Stringów (f-strings)

Najnowocześniejszy sposób wstawiania zmiennych do tekstu.

imie = "Anna"
wiek = 25
punkty = 98.7654

# Podstawowe f-string
print(f"Cześć {imie}, masz {wiek} lat!")

# Formatowanie liczb
print(f"Punkty: {punkty:.2f}")  # Punkty: 98.77

# Wyrównanie tekstu
print(f"|{imie:>10}|")  # "|      Anna|"
print(f"|{imie:<10}|")  # "|Anna      |"
print(f"|{imie:^10}|")  # "|   Anna   |"

4. Wyszukiwanie i Sprawdzanie

email = "user@example.com"

# Czy zawiera?
print("@" in email)  # True

# Pozycja podstringu
print(email.find("@"))  # 4
print(email.index("@"))  # 4 (ale rzuca błąd jeśli nie znajdzie!)

# Sprawdzanie typu
print("123".isdigit())      # True
print("abc".isalpha())      # True
print("abc123".isalnum())   # True
print("HELLO".isupper())    # True

5. 🛠️ Projekt: Walidator Emaila

def waliduj_email(email):
    """Prosta walidacja adresu email"""
    email = email.strip().lower()
    
    # Sprawdź podstawowe warunki
    if "@" not in email:
        return False, "Brak @"
    
    if email.count("@") != 1:
        return False, "Za dużo @"
    
    lokalna, domena = email.split("@")
    
    if not lokalna or not domena:
        return False, "Pusta część przed/po @"
    
    if "." not in domena:
        return False, "Brak kropki w domenie"
    
    return True, "OK"

# Testy
testy = ["user@example.com", "  ADMIN@TEST.PL  ", "brak-małpy", "@@@.com"]
for e in testy:
    ok, msg = waliduj_email(e)
    status = "✅" if ok else "❌"
    print(f"{status} {e.strip():25} → {msg}")
