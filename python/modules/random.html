Moduł Random 🎲

Losowość w Pythonie – od prostych liczb po zaawansowane symulacje.

1. Wstęp do losowości

Moduł random generuje liczby pseudolosowe – czyli takie, które wyglądają na losowe, ale są deterministyczne (można je powtórzyć z tym samym seedem).

import random  # Zawsze na początku!

2. Podstawowe Funkcje

FunkcjaOpisPrzykład
random.random()Liczba float 0.0–1.00.374...
random.randint(a,b)Liczba int z zakresu [a,b]randint(1,6) → 4

import random

print(random.random())          # 0.847...
print(random.randint(1, 10))    # 7
print(random.choice(["tak", "nie"]))  # "nie"

kolory = ["czerwony", "zielony", "niebieski"]
random.shuffle(kolory)
print(kolory)  # np. ['niebieski', 'czerwony', 'zielony']

3. Losowanie z Wagami (choices)

random.choices() pozwala przypisać prawdopodobieństwo każdemu elementowi – idealne do dropów w grach!

import random

przedmioty = ["miecz", "tarcza", "eliksir", "legendarny_klejnot"]
wagi = [50, 30, 19, 1]  # Legendarny dropuje w 1%!

drop = random.choices(przedmioty, weights=wagi, k=1)[0]
print(f"🎁 Dropnął: {drop}")

# Losuj 5 razy
wyniki = random.choices(przedmioty, weights=wagi, k=5)
print(wyniki)

4. Seed – Powtarzalna Losowość

random.seed() ustawia początek generatora. Przydatne do testów i debugowania.

import random

random.seed(42)  # Ustawiamy seed
print(random.randint(1, 100))  # ZAWSZE 82
print(random.randint(1, 100))  # ZAWSZE 15

# Bez seeda – za każdym razem inny wynik
random.seed()  # Reset do losowego seeda

5. 🎁 Projekt: Generator Haseł

Połączmy `random`, `string` i pętle, żeby stworzyć generator mocnych haseł.

import random
import string

def generuj_haslo(dlugosc=12, uzyj_specjalnych=True):
    """Generuje losowe, mocne hasło"""
    znaki = string.ascii_letters + string.digits
    if uzyj_specjalnych:
        znaki += string.punctuation
    
    haslo = ""
    for _ in range(dlugosc):
        haslo += random.choice(znaki)
    
    return haslo

# Użycie
print("🔐 Twoje hasło:", generuj_haslo())
print("🔐 Krótkie hasło:", generuj_haslo(dlugosc=8, uzyj_specjalnych=False))
